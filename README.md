# Pickledbubble

A Sapper app with a Strapi API - will be hosted at pickledbubble.com.

Requires .env file:

* PUBLIC_URL
* BE_CONTAINER_NAME
* FE_CONTAINER_NAME
* MONGO_CONTAINER_NAME
* API_URL
* APP_NAME
* ADMIN_JWT_SECRET
* EMAIL_ADDRESS
* EMAIL_USERNAME
* EMAIL_PASSWORD
* NODE_ENV
* API_PASSWORD
* API_USERNAME
